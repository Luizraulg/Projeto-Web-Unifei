# Sistema de Músicas Web

Aplicação web de reprodução de músicas inspirada em plataformas de streaming como o Spotify, desenvolvida com Node.js e React como parte da disciplina de Desenvolvimento Web da UNIFEI.

## Sobre o projeto

O sistema permite visualizar e reproduzir músicas por meio de uma interface web simples, simulando funcionalidades básicas de serviços de streaming.

## Tecnologias utilizadas

### Backend (desenvolvido por mim)
- Node.js
- Express
- JavaScript

### Frontend
- React
- JavaScript
- CSS

## Funcionalidades

- Listagem de músicas
- Reprodução de áudio
- Interface web interativa
- Integração entre frontend e backend
- Autenticação com uso de tokens

## Minha contribuição

Fui responsável pelo desenvolvimento do backend da aplicação, incluindo:

- Criação do servidor com Node.js e Express
- Implementação das rotas da API
- Estruturação da lógica da aplicação
- Implementação de autenticação utilizando tokens de acesso
- Integração com o frontend

## Como executar o projeto

### Backend

```bash
cd back
npm install
npm start
```
Backend: http://localhost:3001

### Frontend

```bash
cd front
npm install
npm start
```
Frontend: http://localhost:3000





Observações

Este projeto foi desenvolvido em dupla, sendo o backend implementado por mim e o frontend por outro integrante.
